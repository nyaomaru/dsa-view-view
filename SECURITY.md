# Security Policy

## Supported Versions

Security fixes are applied to the latest deployed version of DSA View View.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability. Report it
privately to **nyaonyao0725@gmail.com** with:

- A description of the issue and its potential impact
- Steps or a minimal example that reproduce it
- The affected browser, URL, or commit when known
- Any suggested mitigation

Please avoid accessing other people's data, degrading the service, or publicly
disclosing the issue before a fix can be evaluated.
